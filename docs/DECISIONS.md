# Decisiones de Diseno - PlayStop Backend

## Por que JWT sin estado?
Permite escalabilidad horizontal sin sesiones compartidas.

## Por que BCrypt factor 12?
Balance entre seguridad y rendimiento en el servidor.

## Por que PostgreSQL en Neon Tech?
Plan gratuito generoso con latencia aceptable desde Render.

## Por que Spring Boot 4?
Version mas reciente con soporte para Java 17 y mejor rendimiento de arranque.
