# Glosario de terminos - PlayStop Backend

JWT: Token de autenticacion sin estado firmado con HMAC-SHA256
JPA/Hibernate: Framework de persistencia para mapeo objeto-relacional
DTO: Objeto para transferir datos entre capas de la aplicacion
BCrypt: Algoritmo de hash para contrasenas con factor de trabajo 12
HikariCP: Pool de conexiones JDBC de alto rendimiento para Spring Boot
CORS: Politica que controla acceso cross-origin entre frontend y backend
